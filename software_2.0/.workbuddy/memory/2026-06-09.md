## 实现流式文字+同步语音输出
- `aiResponder-doubao.ts`：新增 `getResponseStream()` 方法，使用 `stream: true` 调用豆包 API
  - 通过 ReadableStream 逐 chunk 回调 `onChunk`
  - 保留 `getResponse()` 非流式接口兼容
- `ttsEngine.ts`：新增流式播放 API
  - `speakStreamStart()`：开启流式模式，清空句子队列
  - `speakStreamPush(text)`：推入文字片段，自动按标点切句，凑齐一句立即送 TTS
  - `speakStreamEnd()`：结束流式，将 buffer 剩余文字强制送出
  - 句子分割正则：`/([。！？；\n.!?;]+)/g`
  - 队列播放：逐句合成+播放，一句播完自动播下一句
- `chat.vue`：`sendText()` 改用 `getResponseStream()`
  - 先添加空 AI 消息，流式填充 content
  - 每个 chunk 同时推入 TTS 句子队列
  - 实时滚动到底部
  - 语音播报开关仍生效：关闭时不推入 TTS
